## Spacing Scale

Declared values (must be multiples of 4):

| Token Name | CSS Variable | Value | Usage |
|------------|-------------|-------|-------|
| xs | `--space-xs` | 4px | Inline icon gaps, micro-gaps |
| sm | `--space-sm` | 8px | Tight element gaps, tag padding |
| md | `--space-md` | 16px | Default element spacing, paragraph gaps |
| lg | `--space-lg` | 24px | Section-inner padding |
| xl | `--space-xl` | 32px | Nav link gaps, card internal spacing |
| 2xl | `--space-2xl` | 48px | Footer padding, major inner breaks |
| 3xl | `--space-3xl` | 64px | Section vertical padding (min) |
| 4xl | `--space-4xl` | 140px | Section vertical padding (max, via clamp) |

Exceptions:
- Section padding uses `clamp(64px, 10vw, 140px)` — fluid between 3xl and 4xl. Source: existing `style.css` `.section` pattern, preserved.
- Container horizontal padding uses `clamp(24px, 4vw, 64px)` — fluid. Source: existing pattern, preserved.
- Nav height: 72px (not on scale — fixed structural value, carried forward from existing implementation).

## Typography

All sizes use `clamp()` for fluid scaling. No breakpoint overrides required (DS-04).

| Role | CSS Variable | Clamp Expression | Weight | Family | Line Height | Letter Spacing |
|------|-------------|-----------------|--------|--------|-------------|----------------|
| Display (h1) | `--text-display` | `clamp(36px, 5.5vw, 72px)` | 300 | Cormorant Garamond | 1.15 | -0.02em |
| Heading (h2) | `--text-heading` | `clamp(28px, 3.5vw, 48px)` | 300 | Cormorant Garamond | 1.15 | -0.01em |
| Subheading (h3) | `--text-subheading` | `clamp(20px, 2vw, 28px)` | 400 | Cormorant Garamond | 1.2 | 0 |
| Body | `--text-body` | 16px (static) | 300 | Jost | 1.75 | 0 |
| Label/Caption | `--text-label` | 13px (static) | 300 | Jost | 1.5 | 0 |
| Eyebrow | `--text-eyebrow` | 10px (static) | 400 | Jost | 1.0 | 0.25em |

**Eyebrow style:** 10px / Jost 400 / uppercase / letter-spacing 0.25em — used for section labels, category tags, nav logo.

**Nav links:** 11px / Jost 400 / uppercase / letter-spacing 0.15em — a sub-variant of the eyebrow role.

Source: CLAUDE.md §Typography, existing `style.css` clamp values carried forward, CONTEXT.md discretion granted for weight/spacing values.

## Color

This phase establishes a deliberately monochromatic palette. Color comes exclusively from photography. (Source: CONTEXT.md D-02)

| Role | CSS Variable | Value | Usage |
|------|-------------|-------|-------|
| Dominant (60%) | `--bg` | `#F8F6F3` | Page background, all surfaces |
| Secondary (30%) | `--bg-surface` | `#F2EFEB` | Subtle card surfaces, placeholder containers |
| Text — Primary | `--text` | `#1A1A1A` | Headings, high-emphasis copy |
| Text — Secondary | `--text-secondary` | `#4A4A4A` | Body copy, paragraph text |
| Text — Tertiary | `--text-tertiary` | `#8A8A8A` | Captions, labels, metadata |
| Accent (10%) | `--accent` | `#1A1A1A` | See reserved-for list below |
| Destructive | n/a | n/a | No destructive actions in this phase |

**Accent (#1A1A1A) reserved for:**
- Hover state of links (text darkens from `--text-secondary` to `--text`)
- Button fill on hover (near-black to full black: `#0D0D0D`)
- Underline on link hover

**No gold, brass, or warm accent colors.** The existing `--gold` / `--gold-light` tokens in `style.css` are replaced entirely. (Source: CONTEXT.md D-02)

**No borders or dividers.** The `--border` token is removed. Whitespace separates sections. (Source: CONTEXT.md D-04)

**Palette character:** Fine stationery — neutral, not warm parchment, not cool white. (Source: CONTEXT.md §Specific Ideas)

## Interaction States

Source: CONTEXT.md D-11 through D-14

| Element | Default | Hover | Focus | Active | Transition |
|---------|---------|-------|-------|--------|-----------|
| Body link | `--text-secondary` (#4A4A4A), no underline | `--text` (#1A1A1A), 0.5px underline appears | `--text` with 2px offset outline | — | 500ms ease |
| Nav link | `--text-tertiary` (#8A8A8A), no underline | `--text-secondary` (#4A4A4A), 0.5px underline expands from left | Same as hover + visible focus ring | — | 500ms ease |
| Button (primary) | `--text` (#1A1A1A) fill, white label | `#0D0D0D` fill (full black) | `#0D0D0D` fill + 2px offset focus ring | `#0D0D0D` | 500ms ease |
| Placeholder container | `--bg-surface` fill, SVG at 30% opacity | No hover state (token phase only) | — | — | — |

**Easing curve:** `cubic-bezier(0.25, 0.46, 0.45, 0.94)` — carried from existing `--ease` token in `style.css`. Smooth deceleration suits luxury timing. (Source: CONTEXT.md discretion granted for easing curves)

**All transitions:** 500ms minimum. "We don't rush." (Source: CONTEXT.md D-14)

**Properties that receive transition timing:**
- `color`
- `background-color`
- `opacity`
- `text-decoration-color`
- `border-color`

## Placeholder System

Source: CONTEXT.md D-08 through D-10, REQUIREMENTS.md IMG-01 / IMG-02

| Variant | CSS Class | Aspect Ratio | SVG Motif | Context |
|---------|-----------|-------------|-----------|---------|
| Hero/Full | `.placeholder--hero` | 16 / 9 | Flowing ballgown silhouette | Full-viewport hero sections |
| Gallery/Portrait | `.placeholder--portrait` | 3 / 4 | Structured fitted gown silhouette | Gallery grids, commission cards |
| Fabric/Square | `.placeholder--fabric` | 1 / 1 | Draped textile fold silhouette | Fabric archive cards |

**Rendering contract:**
- Background: `--bg-surface` (#F2EFEB)
- SVG fill: `--text` (#1A1A1A) at **30% opacity** (rendered via `opacity: 0.30` on the `<svg>` element)
- SVG is centered absolutely within the `aspect-ratio` container
- No text labels inside placeholder containers (removes the `.placeholder-img::after` pattern from existing code)
- No grey blocks, no visible borders

**SVG design intent:** Fashion sketch quality — fine lines, not filled icons. Single continuous stroke per silhouette where possible. (Source: CONTEXT.md §Specific Ideas)

## Token Architecture

Source: CONTEXT.md D-05 through D-07

**File:** `css/tokens.css` (new file, imported first by every HTML page)

Complete token list (~10 tokens as per D-06):

```css
:root {
  /* Color */
  --bg:              #F8F6F3;
  --bg-surface:      #F2EFEB;
  --text:            #1A1A1A;
  --text-secondary:  #4A4A4A;
  --text-tertiary:   #8A8A8A;

  /* Typography */
  --serif:  'Cormorant Garamond', 'Georgia', serif;
  --sans:   'Jost', -apple-system, 'Helvetica Neue', sans-serif;

  /* Easing */
  --ease:   cubic-bezier(0.25, 0.46, 0.45, 0.94);

  /* Transition timing */
  --transition: 500ms var(--ease);
}
```

**Flat naming — no primitive/semantic/component layers.** (Source: CONTEXT.md D-05)

**Font loading:** Google Fonts CDN `<link>` in `<head>` loads Cormorant Garamond (weights 300;400;500;600) and Jost (weights 300;400;500). Replaces current Playfair Display + Inter. (Source: CLAUDE.md §Typography)
