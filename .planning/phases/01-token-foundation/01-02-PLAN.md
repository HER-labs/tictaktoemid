Create directory `img/placeholders/` and three SVG files.

**Design intent:** Fashion sketch quality — fine lines (1-2px stroke), not filled shapes. Single continuous stroke paths where possible. Elegant and minimal, like a couturier's pencil sketch.

**1. `img/placeholders/silhouette-hero.svg`** — Flowing ballgown silhouette
- Viewbox: `0 0 400 225` (16:9 proportion)
- A single flowing ballgown outline: fitted bodice tapering to a waist, then a dramatic full skirt sweeping to the ground with gentle folds
- Stroke-only paths (no fill), stroke width 1.5px
- Stroke color: `#1A1A1A` (the SVG's own color — opacity applied via CSS)
- Include `<title>Dress silhouette placeholder</title>` for accessibility
- No background rectangle — the SVG is transparent

**2. `img/placeholders/silhouette-portrait.svg`** — Structured fitted gown
- Viewbox: `0 0 300 400` (3:4 proportion)
- A column/sheath gown outline: narrow shoulders, fitted through the body, straight or slight mermaid flare at hem
- Stroke-only paths, stroke width 1.5px
- Stroke color: `#1A1A1A`
- Include `<title>Dress silhouette placeholder</title>`

**3. `img/placeholders/silhouette-fabric.svg`** — Draped textile folds
- Viewbox: `0 0 300 300` (1:1 proportion)
- Abstract draped fabric: 3-5 flowing curved lines suggesting silk folds cascading diagonally
- NOT a dress shape — this represents raw material/textile
- Stroke-only paths, stroke width 1.5px
- Stroke color: `#1A1A1A`
- Include `<title>Dress silhouette placeholder</title>`

All SVGs must:
- Use `xmlns="http://www.w3.org/2000/svg"` attribute
- Have `fill="none"` on the root `<svg>` element
- Use `stroke="#1A1A1A"` and `stroke-width="1.5"` on path elements
- Use `stroke-linecap="round"` and `stroke-linejoin="round"` for sketch quality
- Be under 2KB each (simple paths, not complex illustrations)

Add these NEW rules:

```css
/* --- Placeholder System --- */
.placeholder {
  position: relative;
  background: var(--bg-surface);
  overflow: hidden;
  display: flex;
  align-items: center;
  justify-content: center;
}

.placeholder img,
.placeholder svg {
  position: absolute;
  opacity: var(--placeholder-opacity);
  width: 40%;
  height: auto;
}

.placeholder--hero {
  aspect-ratio: 16 / 9;
  width: 100%;
}

.placeholder--portrait {
  aspect-ratio: 3 / 4;
  width: 100%;
}

.placeholder--fabric {
  aspect-ratio: 1 / 1;
  width: 100%;
}
```

     </div> -->
