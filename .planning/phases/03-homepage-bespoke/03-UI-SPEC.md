### New Token Required

Add to `css/tokens.css`:

```css
--border-subtle: rgba(26, 26, 26, 0.08);
```

Accent reserved for: Primary CTA button background (`--text` #1A1A1A on ivory), hero wordmark, clock counter number. These are the only elements that use the darkest value for emphasis. There is NO separate accent color in the monochromatic palette -- contrast and emphasis come from text weight and darkness.

Destructive: Not applicable. No destructive actions exist on these pages.

## Animation Contract

### Hero Entry Sequence (Homepage)

| Step | Element | Animation | Delay | Duration | Easing |
|------|---------|-----------|-------|----------|--------|
| 1 | `.hero-wordmark` | fadeUp (opacity 0->1, translateY 20px->0) | 500ms | 1200ms | `var(--ease)` |
| 2 | `.hero-sub` | fadeUp | 1000ms | 1000ms | `var(--ease)` |
| 3 | `.hero-cta` | fadeUp | 1300ms | 1000ms | `var(--ease)` |
| 4 | `.hero-scroll-hint` | fadeUp | 1800ms | 1000ms | `var(--ease)` |

Hero sequence uses CSS `@keyframes fadeUp` with `animation-fill-mode: forwards`. Initial `opacity: 0` set inline.

### Scroll Reveals (Both Pages)

| Property | Value | Source |
|----------|-------|--------|
| Animation | translateY(30px) -> translateY(0), opacity 0 -> 1 | `.reveal` class in style.css |
| Duration | 800ms | Existing |
| Easing | `var(--ease)` cubic-bezier(0.25, 0.46, 0.45, 0.94) | Existing |
| Trigger | IntersectionObserver at 12% threshold, -40px root margin | main.js |
| Behavior | Fires once, observer unobserves after trigger | Existing |
| Stagger | `.reveal-delay-1` through `.reveal-delay-4` at 100ms increments | Existing |

### Craftsmanship Clock (Homepage)

| Property | Value | Source |
|----------|-------|--------|
| Animation | Count from 0 to current hours | main.js `animateCounter` |
| Duration | 2000ms | Existing |
| Easing | Ease-out cubic: `1 - Math.pow(1 - progress, 3)` | Existing |
| Trigger | IntersectionObserver at 30% threshold | Existing |
| Repeat | Once per session (D-02) — re-animates on new page load only | Context decision |

### Scroll Line Pulse (Homepage Hero)

| Property | Value |
|----------|-------|
| Animation | opacity 0.3 -> 0.8 -> 0.3 |
| Duration | 2000ms |
| Iteration | infinite |
| Easing | ease-in-out |

### GSAP Enhancement Opportunities (Claude's Discretion per D-01)

GSAP v3.13 is loaded via CDN on all pages but not yet used. The executor MAY use GSAP for:

1. **Process step stagger** -- GSAP `stagger` for sequential step reveals instead of CSS delay classes
2. **Chapter entrance** -- GSAP ScrollTrigger for parallax-like image/text reveals on bespoke chapters
3. **Clock number animation** -- GSAP `SplitText` for character-level reveals on the clock number (optional)
4. **Philosophy quote** -- GSAP text reveal, word-by-word or line-by-line (optional)

Constraint: All GSAP enhancements MUST degrade to the existing CSS `.reveal` behavior when GSAP fails to load. The CSS system is the baseline; GSAP is progressive enhancement.

### Reduced Motion

All animations are disabled via existing `@media (prefers-reduced-motion: reduce)` rule in style.css. This applies to both CSS transitions and GSAP (executor must check `prefers-reduced-motion` in JS before GSAP sequences).

## Inline Style Migration Contract (D-04, D-05)

All homepage and bespoke page styles currently live in `<style>` blocks within `<head>`. This phase moves all salvageable styles to `css/style.css` and removes inline blocks entirely.

### Migration Rules

1. Extract all valid CSS from `index.html` inline `<style>` into `css/style.css`
2. Extract all valid CSS from `bespoke.html` inline `<style>` into `css/style.css`
3. Replace every orphaned variable reference per the Color table above
4. Replace every hardcoded rgba gold value per the RGBA table above
5. Remove `<style>` blocks from both HTML files after migration
6. Remove the `.video-badge` element and styles (placeholder for future video, not needed now)
7. Remove the `#0d0905` hardcoded dark gradient in `.page-hero::before` -- replace with `var(--bg-surface)` to `var(--bg)` gradient or remove if hero needs no overlay on ivory
