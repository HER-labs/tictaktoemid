# Phase 2 — UI Design Contract

> Visual and interaction contract for the shared chrome phase: navigation bar, mobile overlay, footer, WhatsApp CTA component, and image directory structure. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (static HTML/CSS/JS on Plesk -- no build tools) |
| Preset | not applicable |
| Component library | none (hand-coded vanilla CSS components) |
| Icon library | none (hamburger built with CSS spans; Instagram icon inline SVG) |
| Font | Cormorant Garamond (serif) + Jost (sans) via Google Fonts CDN |

Declared values from `css/tokens.css` (all multiples of 4):

| Token | Value | Usage in this phase |
|-------|-------|---------------------|

| `--space-md` | 16px | Footer brand margin-bottom, footer column title margin |
| `--space-lg` | 24px | Mobile overlay link gap, footer bottom padding-top, container side padding min |

## Color

Monochromatic palette from Phase 1 tokens. No accent color -- color comes only from photography.

| Role | Value | Usage in this phase |
|------|-------|---------------------|
| Dominant (60%) | `#F8F6F3` (`--bg`) | Page background, nav background (at 85-95% opacity with backdrop-blur) |
| Secondary (30%) | `#F2EFEB` (`--bg-surface`) | Mobile overlay background (at 98% opacity), btn-outline hover fill |
| Text primary | `#1A1A1A` (`--text`) | CTA headline, btn-primary fill, hover state target for links |
| Text secondary | `#4A4A4A` (`--text-secondary`) | Nav wordmark, hamburger bars, btn-outline text, CTA subtext |
| Text tertiary | `#8A8A8A` (`--text-tertiary`) | Nav links default, footer links, footer column titles, footer bottom text |
| Destructive | not applicable | No destructive actions in this phase |

Accent reserved for: Nothing. This is a monochromatic system. The only "accent" is the contrast between `--text` (#1A1A1A) and the ivory background, used for:
- `btn-primary` fill (dark button on light page)
- Hover underline on nav links (0.5px line in `--text`)
- Active nav link indicator

## Interaction States

| Element | Default | Hover | Focus | Active |
|---------|---------|-------|-------|--------|
| Nav links | `--text-tertiary`, no underline | `--text-secondary`, 0.5px underline slides to 100% | Outline: 2px solid `--text-tertiary`, offset 4px | Same as hover |
| Hamburger | 3 horizontal bars | Subtle opacity shift to 0.7 | Outline: 2px solid `--text-tertiary`, offset 4px | Bars transform to X |
| CTA button | `--text` bg, `--bg` text | `#0D0D0D` bg, translateY(-1px) | Outline: 2px solid `--text`, offset 4px | Scale 0.99 |
| Footer links | `--text-tertiary` | `--text` | Outline: 2px solid `--text-tertiary`, offset 4px | Same as hover |
| Instagram link | `--text-tertiary` | `--text` | Outline: 2px solid `--text-tertiary`, offset 4px | Same as hover |

All transitions: 500ms `var(--ease)` (cubic-bezier(0.25, 0.46, 0.45, 0.94)).

Focus styles use `outline` not `box-shadow` for accessibility. Focus-visible only (no ring on click for mouse users):
```css
:focus-visible { outline: 2px solid var(--text-tertiary); outline-offset: 4px; }
```

## Animation Contract

| Animation | Trigger | Duration | Easing | Details |
|-----------|---------|----------|--------|---------|
| Nav scroll state | `window.scrollY > 50` | 500ms | `var(--ease)` | Background opacity transitions from 0.85 to 0.95 (AN-04) |
| Hamburger open | Click/tap toggle | 500ms | `var(--ease)` | Top bar: rotate(45deg) translateY(3px), middle: opacity 0, bottom: rotate(-45deg) translateY(-3px) |
| Mobile overlay | Hamburger toggle | 300ms | `var(--ease)` | Opacity 0 to 1, no slide -- instant position, fade in content |
| CTA banner reveal | Scroll into view | 800ms | `var(--ease)` | `.reveal` class: opacity 0 to 1, translateY(30px) to 0 |
| Link underline | Hover | 500ms | `var(--ease)` | `width: 0` to `width: 100%` on `::after` pseudo-element |

All animations respect `prefers-reduced-motion: reduce`:
```css
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Responsive Contract

| Breakpoint | Changes |
|------------|---------|
| > 768px (desktop) | Horizontal nav links visible, hamburger hidden, footer flex-row |
| <= 768px (mobile) | Nav links hidden, hamburger visible, overlay available, footer stacks vertically, CTA button full-width |

Mobile-first priority per D-17. All shared chrome is designed for mobile first, enhanced for desktop.

## Accessibility Contract

| Requirement | Implementation |
|-------------|----------------|
| Keyboard navigation | All interactive elements reachable via Tab. Escape closes mobile overlay |
| Focus indicators | `:focus-visible` outline on all interactive elements (see Interaction States) |
| ARIA labels | Hamburger: `aria-label="Menu"`, `aria-expanded="false/true"`. Overlay: `role="dialog"`, `aria-label="Navigation"` |
| Skip link | Hidden skip-to-content link as first focusable element: "Skip to main content" |
| Semantic HTML | `<nav>` for navigation, `<footer>` for footer, `<a>` for all links |
| Color contrast | `--text` (#1A1A1A) on `--bg` (#F8F6F3): ratio 14.5:1 (AAA). `--text-tertiary` (#8A8A8A) on `--bg`: ratio 3.2:1 (AA for large text only -- acceptable for 10-11px labels at uppercase with wide tracking) |
| Touch targets | 48px minimum tap height in mobile overlay |
