**Part A — Update js/main.js:**

1. **Line 12:** Change `window.scrollY > 40` to `window.scrollY > 50` (per AN-04 / UI-SPEC scroll threshold).

2. **Lines 17-33:** Replace the entire hamburger toggle block with the closeOverlay pattern. The new code goes inside the existing `if (toggle && navLinks) {` guard:

```javascript
const toggle = document.querySelector('.nav-toggle');
const navLinks = document.querySelector('.nav-links');
if (toggle && navLinks) {
  const spans = toggle.querySelectorAll('span');

  function closeOverlay() {
    navLinks.classList.remove('open');
    toggle.setAttribute('aria-expanded', 'false');
    document.body.style.overflow = '';
    spans[0].style.transform = '';
    spans[1].style.opacity = '';
    spans[2].style.transform = '';
  }

  toggle.addEventListener('click', () => {
    const isOpen = navLinks.classList.contains('open');
    if (isOpen) {
      closeOverlay();
    } else {
      navLinks.classList.add('open');
      toggle.setAttribute('aria-expanded', 'true');
      document.body.style.overflow = 'hidden';
      spans[0].style.transform = 'rotate(45deg) translate(4px, 4px)';
      spans[1].style.opacity = '0';
      spans[2].style.transform = 'rotate(-45deg) translate(4px, -4px)';
    }
  });

  document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape' && navLinks.classList.contains('open')) {
      closeOverlay();
      toggle.focus();
    }
  });
}
```

3. **Lines 107-109:** Update the smooth scroll anchor click handler. Replace:
```javascript
if (navLinks && navLinks.classList.contains('open')) {
  navLinks.classList.remove('open');
}
```
With:
```javascript
if (navLinks && navLinks.classList.contains('open')) {
  closeOverlay();
}
```

Note: `closeOverlay` is defined inside the `if (toggle && navLinks)` block but `navLinks` is declared in the outer scope (line 18). The smooth scroll handler at line 100 is also inside the DOMContentLoaded callback, so it has access to `navLinks`. However, `closeOverlay` is scoped inside the `if` block. To fix this, either: (a) move `closeOverlay` to the outer DOMContentLoaded scope (preferred), or (b) use `navLinks.classList.remove('open'); document.body.style.overflow = '';` inline. Option (a) is cleaner — declare `closeOverlay` at the DOMContentLoaded level, with a null check inside:

```javascript
let closeOverlay = null;
```
...declare at top of DOMContentLoaded, then inside the if block:
```javascript
closeOverlay = function() { ... };
```
...and in the smooth scroll handler:
```javascript
if (navLinks && navLinks.classList.contains('open') && closeOverlay) {
  closeOverlay();
}
```
